##  Project Overview

**AlumniConnect** is a university-focused web platform designed to establish a structured and professional communication channel between alumni and current students. The system enables mentorship, career guidance, job opportunity sharing, and secure messaging within an official environment.

The platform addresses the lack of a centralized alumni network by providing controlled access, role-based features, and administrative moderation.

##  Project Features

### Core Features
- Role-based registration system (Student, Alumni, Admin)
- Admin-controlled alumni verification workflow
- Alumni profile creation and management
- Mentorship request and approval system
- Job and internship posting by alumni
- Secure one-to-one messaging system

### Exclusive Features
- Alumni filtering by department and graduation year
- Mentorship history tracking
- Admin moderation and content control
- Controlled messaging to prevent misuse

---

##  Authentication

- Secure authentication using Laravel Auth
- Role-based access control
- Admin verification of alumni accounts

---

##  CRUD Operations

- **User Profiles:** Create, view, update, and delete accounts
- **Mentorship Requests:** Create, view, accept, or reject requests
- **Job Postings:** Full CRUD operations for job listings
- **Messaging:** Create and read private messages
- **Admin Controls:** Manage users, posts, and activity

---

##  System Workflow

1. Users register and select their role
2. Alumni accounts are verified by the admin
3. Alumni create professional profiles
4. Students browse alumni profiles
5. Students send mentorship requests
6. Alumni accept or reject requests
7. Alumni post job and internship opportunities
8. Students apply and communicate securely
9. Admin monitors all platform activities
